# 2. Especificación

## Requisitos clave

- Landing con hero, navegación superior y dos módulos.
- Modo Campo: campaña, costos, actividades, bitácora y rentabilidad.
- Modo Acopio: ingreso, recepción técnica, muestra visual, análisis IA, castigo y reporte.
- Estados obligatorios: vacío, cargando, éxito y error.
- Datos demo obligatorios: noticias, campaña, lote, imágenes y resultado IA.

## Flujo de demo recomendado

Inicio -> Modo Acopio -> Nuevo ingreso -> Humedad manual -> Muestra visual -> IA -> Castigo -> Reporte.

El Modo Campo debe funcionar como apoyo, pero el flujo de acopio debe ser el más sólido para la presentación.
