|---|---|---|
| / | Landing | Iniciar demo |
| /campo | Dashboard Campo | Abrir módulo |
| /campo/campana | Crear campaña | Guardar campaña |
| /campo/costos | Costos | Guardar costos |
| /campo/actividades | Manejo del cultivo | Guardar actividad |
| /campo/bitacora | Bitácora | Guardar registro |
| /campo/rentabilidad | Rentabilidad | Guardar proyección |
| /acopio | Dashboard Acopio | Nuevo ingreso |
| /acopio/nuevo | Nuevo ingreso | Continuar a recepción |
| /acopio/recepcion/:id | Recepción técnica | Continuar a muestra IA |
| /acopio/muestra/:id | Muestra visual | Analizar con IA |
| /acopio/resultado/:sampleId | Resultado IA | Calcular castigo |
| /acopio/reporte/:id | Reporte | Descargar PDF |
